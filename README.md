## Usage
Open PisitiCoin.py. No external modules required.

Comes with pre populated blockchain. Give it a try!

## Functionalities
  - View a wallet's balance
  - Transfer coins
  - Check blockchain validity
  - Show latest blocks in chain.

## Custom made:
  - Options menu
  - SHA256 implementation
